## Overview

The admin dashboard provides a centralized view of your Okulink Safaris platform, displaying key metrics and quick access to content management features.

## Accessing the Dashboard

The admin dashboard is accessible at `/dashboard` after logging in with your admin credentials. You must have:

- Valid authentication credentials
- Verified email address
- Appropriate admin permissions

## Dashboard Metrics

The dashboard displays real-time counts for all major content types:

### Tours & Bookings
- **Safaris**: Total number of tour packages
- **Bookings**: Customer booking requests
- **Accommodations**: Available lodging options

### Content Management
- **Blogs**: Published blog articles
- **Videos**: Embedded video content
- **Events**: Upcoming and past events
- **Products**: Safari products and merchandise
- **Services**: Available services

### Organization
- **Categories**: Tour categories and subcategories
- **Highlights**: Featured carousel images
- **Enquiries**: Customer inquiry messages

## Quick Actions

From the dashboard, you can quickly navigate to:

- Create new tours
- View and manage bookings
- Add blog posts
- Upload videos
- Manage categories
- View customer enquiries

## Dashboard Features

### Real-Time Updates
All metrics update automatically as content is added, edited, or removed from the system.

### Navigation Shortcuts
Click on any metric card to navigate directly to that content management section.

### Status Overview
Quickly see which areas need attention based on pending bookings or recent enquiries.
