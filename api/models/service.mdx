---
title: Service model
description: Database schema and structure for services
---

The Service model represents services offered by the Okulink Safaris platform.

## Table Structure

The `services` table stores service offerings with pricing and location information.

### Fields

<ResponseField name="id" type="integer">
  Unique identifier for the service
</ResponseField>

<ResponseField name="name" type="string">
  Service name
</ResponseField>

<ResponseField name="description" type="text">
  Detailed service description
</ResponseField>

<ResponseField name="location" type="string">
  Service location
</ResponseField>

<ResponseField name="category" type="string">
  Service category
</ResponseField>

<ResponseField name="price" type="decimal">
  Service price
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly identifier for the service
</ResponseField>

<ResponseField name="image" type="text">
  Primary image URL
</ResponseField>

<ResponseField name="image2" type="string">
  Second image URL (optional)
</ResponseField>

<ResponseField name="image3" type="string">
  Third image URL (optional)
</ResponseField>

<ResponseField name="image4" type="string">
  Fourth image URL (optional)
</ResponseField>

<ResponseField name="image5" type="string">
  Fifth image URL (optional)
</ResponseField>

<ResponseField name="image6" type="string">
  Sixth image URL (optional)
</ResponseField>

<ResponseField name="user_id" type="integer">
  ID of the user who created the service listing
</ResponseField>

<ResponseField name="published" type="integer">
  Publication status (0 = unpublished, 1 = published)
</ResponseField>

<ResponseField name="created_at" type="timestamp">
  Timestamp when the service was created
</ResponseField>

<ResponseField name="updated_at" type="timestamp">
  Timestamp when the service was last updated
</ResponseField>

## Fillable Fields

No fillable fields are explicitly defined in the model.
