Get your Okulink Safaris application up and running quickly with this guide.

## Prerequisites

Before you begin, ensure you have the following installed:

- PHP 8.0.2 or higher
- Composer
- MySQL or compatible database
- Node.js and npm

## Quick Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone <repository-url>
    cd source
    ```
  </Step>

  <Step title="Install PHP dependencies">
    ```bash
    composer install
    ```
  </Step>

  <Step title="Install Node dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Configure environment">
    Copy the example environment file and configure your database settings:
    ```bash
    cp .env.example .env
    ```
    
    Edit `.env` and set your database credentials:
    ```
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=your_database_name
    DB_USERNAME=your_username
    DB_PASSWORD=your_password
    ```
  </Step>

  <Step title="Generate application key">
    ```bash
    php artisan key:generate
    ```
  </Step>

  <Step title="Run database migrations">
    ```bash
    php artisan migrate
    ```
  </Step>

  <Step title="Create storage link">
    ```bash
    php artisan storage:link
    ```
  </Step>

  <Step title="Build frontend assets">
    ```bash
    npm run dev
    ```
  </Step>

  <Step title="Start the development server">
    ```bash
    php artisan serve
    ```
    
    Your application will be available at `http://localhost:8000`
  </Step>
</Steps>

## First Steps

Once your application is running, you can:

1. **Register an account** - Visit `/register` to create a new user account
2. **Browse safaris** - Explore available tours and safari packages at `/safaris`
3. **Make a booking** - Select a tour and create your first booking
